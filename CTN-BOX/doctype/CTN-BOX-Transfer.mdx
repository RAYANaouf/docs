The **CTN-BOX Transfer** Doctype is used to record and manage the movement of boxes (`CTN-BOX`) from one place to another within the system.

## ðŸšš Purpose

This Doctype allows users to:

- Log transfers of one or more boxes between locations.
- Track source and destination places.
- Maintain a history of box movements.
- Ensure accountability of box handling across departments or warehouses.

## ðŸ§© Fields

| Field Name        | Type        | Description                                          |
|------------------|-------------|------------------------------------------------------|
| `transfer_date`  | Date        | Date when the transfer takes place                  |
| `from_place`     | Link        | Source place (linked to `CTN-Place`)                |
| `to_place`       | Link        | Destination place (linked to `CTN-Place`)           |
| `boxes`          | Table       | List of boxes being transferred                     |
| `status`         | Select      | Transfer status (e.g., Draft, In Progress, Completed) |
| `remarks`        | Text        | Optional notes or observations                      |

> The `boxes` table links to the **CTN-BOX Transfer Item** child table.
