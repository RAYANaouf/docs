- Displays a list of **POS Check In / Check Out**.
- Includes:
  - date time
  - amount
  - type (in / out)
  - Cashier/user name
- You can **filter** the list by:
  - type (in / out)

- Shows detailed information about the selected Check In / Out, including:
  - Total cash in/Out
  - date and time
  - type (in / out)
  - Cashier/user name
  - Reason (as a note or reason to response to the question "Why?")
