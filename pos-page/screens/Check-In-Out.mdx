The **Check In / Out** screen is designed to quickly access and review your latest POS transactions.

---

#### ðŸ”¹ Check In / Out List Section

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/checkInOutList.png"
/>

- Displays a list of **POS Check In / Check Out**.
- Includes:
  - date time
  - amount
  - type (in / out)
  - Cashier/user name
- You can **filter** the list by:
  - type (in / out)

#### ðŸ”¸ Check In Out Detail Section

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/CheckInOutDetails.png"
/>

- Shows detailed information about the selected Check In / Out, including:
  - Total cash in/Out
  - date and time
  - type (in / out)
  - Cashier/user name
  - Reason (as a note or reason to response to the question "Why?")

---



#### ðŸ”„ Use Case

This screen is essential for:

- Registering **manual cash movements** not tied to a sale (e.g., starting cash, withdrawals, deposits, refunds)
- Maintaining **accurate cash drawer balances** throughout the day
- Tracking **who performed each cash movement** and why
- Improving **financial transparency** and **accountability** within POS operations
